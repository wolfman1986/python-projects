# Calculator Project

This is a simple Python calculator application built with both a command-line interface and a graphical user interface.

## Features
- Basic arithmetic operations
- Command-line input handling
- GUI built with Tkinter
- Clear button and result display
- Error handling for invalid input

## How It Works
The command-line version prompts the user to choose an operation and enter two numbers. It then calculates and displays the result.

The GUI version opens a calculator window with buttons for numbers and operators. Users click buttons to build an expression, then press `=` to calculate the answer.

## Technologies Used
- Python
- Tkinter

## Notes
The GUI version uses Python's `eval()` function to evaluate expressions entered in the calculator window.
